# Todoapp with Angular and Django Rest 
Made by:
- Django REST
- Angular

Features:
- add lists,tasks,daily tasks
- daily refresh of daily tasks
- completed tasks with date of creation and completion
- tasks sorted by creation
- daily tasks sorted by time
